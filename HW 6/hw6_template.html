Natural Breaks Classification

This classification breaks the data into distinct points using an algorithm. One benefit is that it accurately represents the data by identifying true clusters and classes. A disadvantage, however, is that the number ranges can be very large and specific to the individual dataset, making it difficult to use in comparing multiple maps with one another.

I chose this ratio because I was interested in the income distribution at the lowest tier of collected data.

Equal Interval Classification

This classification divides the data into equal size groups. It is very easy to interpret on a legend, and can be used for presenting to non-technichal audiences. Sometimes though, the data will fall into only one or two categories, leaving the others blank.

I chose this ratio because I wanted to see the income distribution around the mean salary for Americans.

Quantile Classification

This classification divides the data into classes with an equal number of data points. It is good for emphasizing the relative position of the data, and for comparing medians. Two drawbacks are 1) it can be hard to see clusters in the data and 2) features in the same class may differ widely, particularly when the data is not evenly distributed across its range.

I chose this ratio, like the first map, because I wanted to see where the highest earners in the state were concentrated.
